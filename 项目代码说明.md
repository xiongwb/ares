# 项目代码说明

## react native 代码目录说明

```
ares
| -- app
     |-- assets        # => 存放图片的文件夹
     |-- components    # => 存放自己封装的通用组件的文件夹
     |-- features      # => 存放了实现二维码等功能时的一些实验代码的文件夹，暂时没有用到
     |-- pages         # => 存放了页面组件，一个文件代表一个页面
     |-- app.js        # => 路由文件
| -- index.android.js  # => android 平台的入口文件
| -- index.ios.js      # => ios 平台的入口文件
```
